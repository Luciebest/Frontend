Don’t know how this works?

Admin

Here’s what you can do:
-create, edit, delete events about the campus (see Events section from the sidebar)
-approve/decline topics submitted by students (see Notifications section)
Keep in mind that:
-the Reviews section and the Topics section are only for viewing, nothing can be added or modified
-you can view more details about a certain topic/event by simply clicking on it
-you can’t add likes/comments. Let the students be the judges here :)

Student

Here’s what you can do:
-add likes & comments on topics about the campus (see Topics section from the sidebar). Also, you can create & submit for approval different topics
-add likes on Events (see Events section)
-see if the Administrator has approved/declined the topic(s) submitted by you (see Notifications section)
-sort by number of likes/most recent post
Keep in mind that:
-you can view more details about a certain topic/event by simply clicking on it
